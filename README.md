# Weather Adaptive Road Mapping

A confidence-based multi-sensor fusion framework for autonomous vehicle perception under varying weather conditions.

## Features

- Weather Detection
- Image Enhancement
- YOLO Object Detection
- Depth Estimation
- Adaptive Sensor Fusion
- Object Trackinggit 
- Risk Assessment
- ADAS Decision Engine
- HUD Visualization

## Project Structure

- weather/ : weather detection and enhancement
- models/ : detection, tracking, depth estimation and fusion
- adas/ : risk analysis and collision estimation
- ui/ : dashboard and visualization
- config/ : project settings
